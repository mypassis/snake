# Neon Snake

A sleek, mobile-optimized Snake game featuring a modern neon aesthetic and responsive touch controls. Built entirely with vanilla HTML, CSS, and JavaScript.

## 📖 Description
Neon Snake reimagines the classic arcade experience with a contemporary UI. Designed from the ground up for mobile devices, it offers both swipe gestures and on-screen directional buttons for seamless gameplay. The game tracks your current score and persists your high score locally across sessions.

## ✨ Features
- **📱 Mobile-First Design**: Fully responsive layout optimized for touchscreens and varying viewport sizes
- **🎮 Dual Controls**: Intuitive swipe gestures + on-screen D-pad buttons for accessible gameplay
- **✨ Modern UI/UX**: Glassmorphism overlays, neon glow effects, and smooth CSS transitions
- **⚡ Zero Dependencies**: Pure vanilla HTML, CSS, and JavaScript—no frameworks or build steps required
- **💾 Persistent High Scores**: Automatically saves and loads your best run using `localStorage`
- **🌙 Dark Theme**: Easy-on-the-eyes color palette with vibrant indigo, violet, and fuchsia accents

## 🚀 How to Run
1. Clone this repository or download the source files (`index.html`, `style.css`, `script.js`).
2. Navigate to the project directory.
3. Open `index.html` directly in any modern web browser.
4. For the optimal experience, enable mobile emulation in Chrome DevTools or open the file on a physical smartphone/tablet.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: CSS Custom Properties, Flexbox, Grid, Backdrop Filters, CSS Animations
- **Storage**: `localStorage` for persistent high scores
- **Fonts**: Inter (UI), JetBrains Mono (Scores/Labels)
