# Design System

## Aesthetics
Modern, sleek, and visually striking. NEVER use default browser styles.

## Colors
Use a sophisticated palette. Prefer dark themes (e.g., #0f172a background, #f1f5f9 text) with vibrant, glowing accents (indigo, violet, fuchsia). Use CSS variables.

## Typography
Use 'Inter' for sans-serif and 'JetBrains Mono' for code. Large, bold headings (clamp up to 4rem) with tight tracking. Muted, readable body text.

## Layout
Generous whitespace (padding: 2rem+). Centered max-width containers (1200px). Fully responsive Flexbox/Grid.

## Components
Backgrounds should be subtle (rgba). Use 1px solid borders with low opacity. Border-radius 12px-16px. Use backdrop-filter: blur() for glassmorphism.

## Interactions
Smooth transitions (0.3s ease). Hover states should scale slightly (1.02) and increase glow/shadow.
